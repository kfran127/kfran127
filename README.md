## Cloud Security

- [AWS Cloud Security Lab](https://github.com/kfran127/aws-cloud-security)  
  Custom VPC architecture, IAM least-privilege enforcement, CloudTrail 
  logging, and misconfiguration detection and remediation on AWS.

- [Cloud Security Scout](https://github.com/kfran127/cloud-security-scout) *(In Progress)*  
  Automated AWS misconfiguration scanner using Lambda and Boto3. 
  Detects public S3 buckets, open security groups, and unencrypted 
  RDS instances. Sends real-time alerts via SNS. Results stored in 
  DynamoDB for trend tracking.

- [Secure File Vault](https://github.com/kfran127/secure-file-vault) *(In Progress)*  
  Secure file sharing system built on AWS using S3 with KMS encryption, 
  IAM least-privilege roles, Cognito authentication, and CloudTrail 
  audit logging.

- [AWS IAM Security Automation](https://github.com/kfran127/aws-iam-security-automation) *(In Progress)*  
  Python and Boto3 scripts to automate IAM security audits — detects 
  unused credentials, overly permissive policies, and users missing 
  MFA. Generates remediation reports and sends alerts via SNS.

- [AWS Security Lab — Terraform](https://github.com/kfran127/aws-security-lab-terraform) *(In Progress)*  
  Infrastructure as Code rebuild of the AWS Cloud Security Lab using 
  Terraform. Deploys VPC, subnets, security groups, EC2, S3, and IAM 
  roles as versioned, auditable code. Demonstrates secure-by-default 
  cloud infrastructure design.
